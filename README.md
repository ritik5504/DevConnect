# 🔐 Authentication System (Node.js + Express + MongoDB)

## 🚀 Features
- User Registration
- User Login
- JWT Authentication
- Protected Routes
- Password Hashing (bcrypt)

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT
- bcrypt

## 📌 API Endpoints

### Register
POST /api/auth/register

### Login
POST /api/auth/login

### Get User
GET /api/auth/me

## ⚙️ Setup

1. Clone repo
2. Install dependencies
   npm install
3. Create `.env` file
4. Run server
   node server.js
